Aturan Umum

Bagi mahasiswa Angkatan 2019 wajib memenuhi Mata Kuliah Pilihan Wajib Profil (MKPWP) dengan kententuan jumlah SKS >= 6 SKS atau setara dengan 2 mata kuliah (jika per mata kuliah adalah 3 SKS). Untuk lebih lengkapnya, dapat dilihat pada halaman profil profesional mandiri . done

  1. Mahasiswa mengajukan permohonan yudisium melalui aplikasi Yudisium done
  2. Mahasiswa melengkapi berkas-berkas persyaratan yudisium melalui aplikasi Yudisium. done
  3. Mahasiswa berkoordinasi dengan administrasi FTI terkait persyaratan dari Prodi Informatika. done
  4. Setiap unit (Biro 2, Biro 3, Perpustakaan, PPB, Prodi Informatika) melakukan validasi lewat aplikasi Yudisium.
  1. Lembar persetujuan revisi laporan Skripsi
  2. Lembar pengesahan laporan Skripsi
  3. Ceklist mata kulilah Prodi Informatika
  4. Bebas pinjaman dari Biro 2
  5. Bebas pinjaman dari Biro 3
  6. Memenuhi poin keaktivan(5) yang disyaratkan bidang kemahasiswaan (WD3 dan Biro 3)
  7. Mengunggah laporan dan program ke aplikasi SINTA (Perpustakaan)
  8. Lulus program ICE (Introduction to College English) (PPB)
  9. Mengumpulkan artikel jurnal Skripsi ke Lab FTI UKDW atau FTI
  10. Melakukan pembayaran Yudisium (Prodi / FTI)
